BAU-Enhancement-Liability-Agreed rate of exchange- update field type to decimals & add new field for comments



Description

Currently, Agreed Rate of Exchange is a string field. We need to change this field to decimal (this is needed because of ACW changes as we are required to send this field to ODS>ZIP). For e.g. 0.72653 

We need 2 changes as part of this JIRA-

Agreed rate of exchange field should be changed to accept numbers only (with decimal). 

Add new field “Agree Rate of Exchange Additional Comments” on the screen just under Agreed Rate of Exchange so that user can add text in this field. This is optional field. This needs to be printed on Subjectivities page, just under Agreed rate of Exchange field.
